# Beginner AI Course

## Your Journey into AI Development

Welcome to your beginner-friendly AI learning course! This course takes you from zero to building your own AI applications through 10 hands-on lessons.

---

## Prerequisites

Before starting, make sure you have:
- Python installed (3.10+)
- A `.env` file with your OpenAI API key (`OPENAI_API_KEY=sk-...`)
- Jupyter notebook support (VS Code with Jupyter extension or JupyterLab)

## Lesson Descriptions

### Lesson 1: What is AI?
*No coding required* - Learn the fundamentals of AI, Machine Learning, and LLMs through simple explanations and analogies.

### Lesson 2: Your First AI Call
Connect to OpenAI's API and make your first programmatic AI call. Build a joke generator!

### Lesson 3: The Art of Prompts
Learn prompt engineering - how to write prompts that get great results. Build a story generator.

### Lesson 4: Conversations
Understand how AI "remembers" context. Build a Memory Bot that maintains conversation history.

### Lesson 5: System Prompts
Give AI a personality! Create different personas: pirate, teacher, and chef.

### Lesson 6: Temperature
Control creativity vs consistency. Build a comparison tool to see the difference.

### Lesson 7: Structured Output
Get AI to respond in specific formats like JSON. Build a recipe extractor.

### Lesson 8: Multi-Step Workflows
Chain multiple AI calls together. Build a complete blog post generator.

### Lesson 9: AI Agents
Build your first AI agent that can think, plan, and act. Create a research assistant.

### Lesson 10: Final Project
Combine everything to build a Personal AI Tutor that can teach any topic!

## Troubleshooting

### API Key Issues
- Make sure your `.env` file exists in the project root
- The key should start with `sk-`
- Run `load_dotenv(override=True)` to reload

### Import Errors
- Ensure you've selected the right Python kernel
- Install required packages: `pip install openai python-dotenv`

### Rate Limits
- OpenAI has rate limits on API calls
- Wait a moment and try again
- Consider using `gpt-4o-mini` for cheaper/faster calls
